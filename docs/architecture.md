# 企业客服系统 — 系统架构说明

## 项目概述

企业客服系统是基于 SpringCloud Alibaba 微服务架构的在线客服平台，提供实时消息通信、客户管理、服务监控等功能。

## 技术架构

```
┌─────────────────────────────────────────────────────┐
│                    前端层 (HTML + JS + CSS)           │
└─────────────────────────┬───────────────────────────┘
                          │ HTTP / WebSocket
┌─────────────────────────▼───────────────────────────┐
│                Gateway (SpringCloud Gateway)         │
│              路由转发 · 鉴权 · 限流                    │
└─────────────────────────┬───────────────────────────┘
                          │
┌─────────────────────────▼───────────────────────────┐
│              Nacos (服务注册 · 配置中心)               │
└─────────────────────────┬───────────────────────────┘
                          │
┌─────────────────────────▼───────────────────────────┐
│                  chat-service (业务服务)              │
│           ┌──────────┬──────────┬──────────┐        │
│           │ 用户管理  │ 客服聊天  │ 消息管理  │        │
│           └──────────┴──────────┴──────────┘        │
│           ┌──────────────────────┐                   │
│           │  Sentinel (流量控制)   │                   │
│           └──────────────────────┘                   │
└──────────────┬─────────────────┬────────────────────┘
               │                 │
┌──────────────▼────────┐  ┌─────▼──────────────────┐
│       MySQL 5.7+      │  │      Redis 6.0+        │
│   (持久化 · 业务数据)    │  │  (缓存 · 会话 · 消息队列) │
└───────────────────────┘  └────────────────────────┘
```

## 模块划分

| 模块 | 职责 | 端口 |
|------|------|------|
| chat-common | 公共实体、工具类、统一响应 | — |
| chat-gateway | 网关入口、路由转发、鉴权 | 8080 |
| chat-service | 核心业务逻辑 | 8081 |

## 技术选型

| 组件 | 版本 | 说明 |
|------|------|------|
| SpringBoot | 2.7.x | 基础框架 |
| SpringCloud Alibaba | 2021.0.x | 微服务套件 |
| Nacos | 2.1.x | 注册中心 & 配置中心 |
| Sentinel | 1.8.x | 流量控制 & 熔断降级 |
| SpringCloud Gateway | 3.1.x | API 网关 |
| MySQL | 5.7+ | 关系型数据库 |
| Redis | 6.0+ | 缓存 & 会话管理 |
| MyBatis-Plus | 3.5.x | ORM 框架 |
